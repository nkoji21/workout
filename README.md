# workout

自分専用AIパーソナルトレーナーのコンテキスト基盤。

トレーニング記録・身体ステータス・目標を蓄積し、「今日何したら良い？」と聞くだけで、その時点の自分に最適な提案を返してもらうためのリポジトリ。

## コンセプト

このリポジトリは「筋トレの記録保管庫」ではなく、**AIパーソナルトレーナーに渡すコンテキスト基盤**として設計されています。

AIは毎回メニューを出す機械ではなく、本物のトレーナーとして振る舞います。
- 今日は重くいくべきか、軽くすべきか
- 今日は休むべきか
- どの部位を優先すべきか

…をログ・疲労・目標をもとに自律的に判断します。

## 使い方

| やること | 方法 |
|---------|------|
| 今日どうすべか判断してほしい | 「今日何したら良い？」と聞く |
| トレ後の記録 | 「終わった」と言うだけ（差分があれば一言添える） |
| 週次振り返り | 「今週の振り返りして」と言う |

## ファイル構成

```
profile/        身体データ・目標・ジム設備
status/         現在の疲労状態・種目別進捗
logs/           日次ログ・週次振り返り
plans/          次回予定メニュー
templates/      ログのテンプレート
.claude/skills/ AIスキル定義（today / log / weekly）
CLAUDE.md       AIの振る舞い・判断基準
```
